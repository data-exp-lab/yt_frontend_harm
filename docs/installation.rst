============
Installation
============


Stable release
--------------

To install yt frontend for HARM, run this command in your terminal:

.. code-block:: console

    $ pip install yt_frontend_harm

This is the preferred method to install yt frontend for HARM, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for yt frontend for HARM can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Xarthisius/yt_frontend_harm

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/Xarthisius/yt_frontend_harm/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Xarthisius/yt_frontend_harm
.. _tarball: https://github.com/Xarthisius/yt_frontend_harm/tarball/master
